# Publication and archival status

SotA Lens is distributed as open-source research software and archived through Zenodo.

## Current status

- **Software DOI:** [10.5281/zenodo.19860899](https://doi.org/10.5281/zenodo.19860899).
- **Rendered preprint PDF:** available through the project website while arXiv endorsement/submission is pending.
- **arXiv:** source is available under `arxiv/`; the arXiv identifier should be added after endorsement/submission.
- **JOSS:** a JOSS-ready manuscript is available under `paper/`, but the project is not currently under JOSS review.

The initial JOSS submission was [procedurally rejected](https://github.com/openjournals/joss-reviews/issues/10479#issuecomment-4336662411) because JOSS currently requires at least six months of public open-development history for projects developed privately before release. This was not a rejection of the manuscript build, license, or software quality.
