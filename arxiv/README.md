# arXiv preprint

This directory contains the longer methodology preprint for SotA Lens.

Build locally:

```bash
cd arxiv
make
```

Files:

- `main.tex` — full preprint source.
- `references.bib` — BibTeX bibliography.
- `figures/map_of_field.pdf` — field map used in the proof-of-concept case study.
- `figures/sota_article_net.png` — DPM/SAR citation-network figure.
- `Makefile` — reproducible PDF build.

Recommended arXiv category: `cs.DL` with possible cross-listing to `cs.IR`.
