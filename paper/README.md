# Paper

This directory contains the **JOSS-ready manuscript** for SotA Lens.

- Main manuscript: `paper.md`
- Bibliography: `paper.bib`

The manuscript is written in the format expected by the Journal of Open Source Software (JOSS), but the project is **not currently under JOSS review**. The first submission was procedurally rejected because the repository did not yet have the public open-development history JOSS currently requires for projects developed privately before release.
